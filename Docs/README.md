#  Atom Loves You Tailwind 3.0 Audio Player

A Pen created on CodePen.io. Original URL: [https://codepen.io/Khristin-Schenk/pen/poOZdmG/df4db54120f632b6e48608636de44162](https://codepen.io/Khristin-Schenk/pen/poOZdmG/df4db54120f632b6e48608636de44162).

Saw this amazing player design by Steve Schoger (https://twitter.com/steveschoger/status/1499780794193166341?s=20&t=SC8akXhJOIdDj4AjnASuAA) and asked if we could use it as an example for AmplitudeJS. He was super gracious and let us make a sample player. It's a perfect use case for AmplitudeJS showing the power of the functionality along with the flexibility of different designs.

The player is built with the newest version of TailwindCSS v3.0 and AmplitudeJS 5.2
